Este projeto é uma página estática inspirada em plataformas de vídeo como o YouTube.
Ela inclui um cabeçalho com busca e ícones de usuário, um menu lateral com navegação por seções, uma área principal com vários vídeos incorporados do YouTube e um rodapé organizado por temas de estudo (programação, data science, UX, mobile etc.).
O foco é praticar HTML e CSS (especialmente layout com Flexbox) criando uma interface moderna e próxima de um site real de streaming de vídeos.
